## 📝 使用说明

### 启动应用
```bash
# 启动后端
node server/index.js

# 启动前端 (新终端)
cd client && npm start
```

### 使用流程
1. 访问 http://localhost:3000
2. 系统自动清空历史分析数据
3. 输入12个月的经营数据
4. 点击"提交数据并分析"
5. 系统自动跳转到分析页面
6. 查看分析结果和图表
7. 获取AI建议
